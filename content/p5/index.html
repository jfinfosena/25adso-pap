    Clase 5: MГіdulos en FastAPI

    En FastAPI, los mГіdulos son una forma de organizar y estructurar el cГіdigo de manera eficiente, promoviendo la modularidad, reutilizaciГіn y escalabilidad. Esto se logra principalmente mediante el uso de APIRouter y la organizaciГіn de archivos en un proyecto. A continuaciГіn, te explico cГіmo se manejan los mГіdulos en FastAPI y cГіmo estructurar un proyecto modular, basado en las mejores prГЎcticas y la informaciГіn disponible:

    1. Uso de APIRouter para modularizar rutas

    FastAPI proporciona la clase APIRouter para dividir las rutas de la API en mГіdulos mГЎs pequeГұos y manejables. Esto permite separar la lГіgica de las rutas en diferentes archivos, lo que facilita el mantenimiento y la colaboraciГіn en equipo.

    Ejemplo bГЎsico de modularizaciГіn con APIRouter:

    1. Crear un archivo para un mГіdulo especГӯfico (por ejemplo, routers/users.py): 

      from fastapi import APIRouter

@router.get("/users/")
async def read_users():
    return [{"username": "user1"}, {"username": "user2"}]

@router.post("/users/")
async def create_user(user: dict):
    return {"message": "User created", "user": user}

    2. Integrar el mГіdulo en el archivo principal (por ejemplo, main.py):

from routers import users

app = FastAPI()

# Incluir el router de users
app.include_router(users.router, prefix="/api", tags=["users"])

@app.get("/")
async def root():
    return {"message": "Welcome to FastAPI"}

    3. prefix: Define un prefijo para todas las rutas del mГіdulo (por ejemplo, /api/users/).

    4. tags: Agrupa las rutas en la documentaciГіn automГЎtica (Swagger UI) para mejor organizaciГіn.

    5. Ejecutar la aplicaciГіn: 

      fastapi dev main.py
      Esto permite que las rutas definidas en users.py estГ©n disponibles bajo el prefijo /api/users/.

    3. MГіdulos importantes instalados con FastAPI

    Al instalar FastAPI (pip install "fastapi[standard]"), se incluyen mГіdulos clave que potencian su funcionalidad:

    • Pydantic: Permite la creaciГіn de modelos para validaciГіn de datos y serializaciГіn. Se usa para definir esquemas (schemas) que validan los datos de entrada y salida. 

      from pydantic import BaseModel

    username: str
    email: str

    • Starlette: Framework ASGI ligero que proporciona la base para servicios web asГӯncronos en FastAPI.

    AdemГЎs, puedes extender FastAPI con otros mГіdulos como: - SQLAlchemy: Para manejar bases de datos relacionales. - Jinja2: Para motores de plantillas si necesitas renderizar pГЎginas web. - python-jose o passlib: Para autenticaciГіn y manejo de tokens.

    4. Principios clave para la modularizaciГіn

    Para mantener un proyecto FastAPI modular y escalable, sigue estas prГЎcticas: - SeparaciГіn de intereses: Divide la lГіgica de negocio, rutas, modelos y esquemas en mГіdulos separados. - InyecciГіn de dependencias: Usa fastapi.Depends para desacoplar componentes y mejorar la testabilidad.

async def get_current_user():
    return {"user_id": 1}

    return user
    - Testabilidad: Estructura el cГіdigo para facilitar pruebas unitarias, usando mocks y dependencias. - ReutilizaciГіn: Crea mГіdulos reutilizables para lГіgica comГәn (por ejemplo, autenticaciГіn, validaciones).
