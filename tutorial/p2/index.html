    üìò Ciclos en Python

    1. Introducci√≥n

    Los ciclos (o bucles) permiten ejecutar un bloque de c√≥digo repetidamente sin necesidad de reescribirlo.
    En Python existen dos construcciones b√°sicas:

    Construcci√≥n Uso t√≠pico ¬øTermina autom√°ticamente?
    for Recorrer iterables S√≠, cuando el iterable se agota
    while Repetir mientras se cumpla una condici√≥n No, puede ser infinito si no se actualiza la condici√≥n

    A su vez, cada construcci√≥n admite flujos de control (break, continue, else) y combinaciones avanzadas (else, else + continue, else + break).

    2. Ciclo for

    2.1 Sintaxis b√°sica

    for variable in iterable:
    # bloque a repetir
else:          # opcional
    # se ejecuta si NO hubo break

    2.2 Objetos iterables m√°s usados

    Iterable Ejemplo ¬øQu√© recorre?
    range() range(5) N√∫meros enteros
    list [1, 2, 3] Elementos de la lista
    str "abc" Caracteres
    dict {"a": 1, "b": 2} Claves (por defecto)
    enumerate() enumerate(["x", "y"]) Pares (√≠ndice, elemento)
    zip() zip(l1, l2) Tuplas emparejadas

    2.3 Ejemplos paso a paso

    2.3.1 Recorrer un rango

    2.3.3 Recorrer diccionarios

    2.3.4 Usar enumerate para √≠ndice + valor

    2.3.6 Bucle anidado (nested for)

    2.3.7 Comprensi√≥n de listas (for en una sola l√≠nea)
